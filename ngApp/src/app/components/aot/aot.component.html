AOT ahead of time compiling

faster rending: with AOT, the brower downloads the pre-compiled version of the app, the brower render the application immediately without waiting to compile the app first

Fewer asynchronous requests: the compiler inlines the external html and CSS into the JS file, eliminating separate ajax requests for those files

Detect template errors earlier

Better security: The compiler compiles HTML templates and components into js files. it minimise the risks for injection attacks
